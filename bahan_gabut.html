Apa Itu HTML Semantik?

HTML Semantik adalah penggunaan elemen HTML yang memiliki makna atau arti yang jelas, bukan hanya sekadar untuk tampilan visual. Dengan menggunakan elemen semantik, kode HTML kita menjadi lebih mudah dibaca oleh manusia maupun mesin seperti mesin pencari dan screen reader.

Contoh elemen semantik yang sering digunakan antara lain <header>, <nav>, <main>, <article>, <aside>, dan <footer>. Masing-masing elemen ini memiliki fungsi dan makna yang spesifik dalam struktur halaman web.

Keuntungan Menggunakan Semantic HTML

Penggunaan HTML semantik memberikan banyak manfaat: meningkatkan aksesibilitas bagi pengguna dengan disabilitas, membantu mesin pencari memahami konten halaman (SEO), membuat kode lebih mudah dipelihara, dan meningkatkan keterbacaan kode bagi developer lain.

<!-- Struktur Semantic HTML -->
<header>...Judul dan Navigasi...</header>
<main>
  <article>...Konten Utama...</article>
  <aside>...Konten Pendukung...</aside>
</main>
<footer>...Footer...</footer>

Flexbox: Layout Modern yang Responsif

CSS Flexbox (Flexible Box Layout) adalah metode tata letak satu dimensi yang dirancang untuk mengatur elemen dalam satu baris atau kolom. Flexbox memudahkan kita dalam membuat layout yang responsif tanpa menggunakan properti float yang lebih kompleks dan sulit dikontrol.

Properti Utama Flexbox

Untuk mengaktifkan flexbox, cukup tambahkan display: flex pada elemen kontainer (flex container). Selanjutnya, elemen-elemen di dalamnya (flex items) akan otomatis tersusun dalam satu baris secara horizontal.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
  flex-wrap: wrap;
}

Properti justify-content mengatur distribusi item secara horizontal, sedangkan align-items mengatur posisi vertikal. Dengan kombinasi keduanya, kita bisa membuat layout yang sangat fleksibel dan responsif di berbagai ukuran layar.

display: flex flex-direction justify-content align-items flex-wrap gap flex-grow flex-shrink

Styling dengan CSS: Dari Dasar ke Lanjutan

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan format dokumen HTML. Dengan CSS, kita bisa mengontrol warna, tipografi, spacing, animasi, dan banyak aspek visual lainnya dari sebuah halaman web.

CSS Variables untuk Konsistensi Desain

Salah satu fitur modern CSS yang sangat berguna adalah CSS Custom Properties atau biasa disebut CSS Variables. Dengan mendefinisikan nilai seperti warna dan font sekali di :root, kita bisa menggunakannya di seluruh stylesheet dan memudahkan perubahan tema di masa depan.

:root {
  --primary-color: #1a1a2e;
  --accent-color: #e94560;
  --font-size-base: 1rem;
}

.button {
  background: var(--accent-color);
  font-size: var(--font-size-base);
}

Dengan pendekatan ini, mengubah seluruh palet warna website cukup dilakukan dengan mengedit beberapa baris di bagian :root saja — sangat efisien dan mudah dipelihara dalam jangka panjang.
